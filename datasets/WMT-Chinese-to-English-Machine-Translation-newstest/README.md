# WMT中英机器翻译新闻测试集

## 数据集描述
中英机器翻译新闻领域测试集，用于评估模型翻译质量。

### 数据集简介
该数据集来源于世界翻译大会[WMT 2021](https://www.statmt.org/wmt21/translation-task.html) News Translation Task，聚焦于新闻文本，包括WMT18、WMT19、WMT20三个版本，共计3960个平行句子对。

### 数据集支持的任务
中英语向机器翻译质量评估

## 数据集的格式和结构

### 数据格式
文本格式，source和reference逐行对齐
